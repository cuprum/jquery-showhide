JQuery ShowHide plugin

Показ/cкрытие блоков с запоминаем состояния.

Для работы плагина необходимо включить поддержку JSON:


$.cookie.json = true; //включение обрабоки JSON в куках.

  • Вызов плагина без параметров.

    Клик отслеживается на первом дочернем элементе внутри блока. Aнимационный эффект slideUp/slideDown со временем анимации 400 мс применяется ко второму дочернему элементу блока. Имя куки устанавливается равным адресу текущей страницы (имя хоста + путь).

    
<div class="block1">
	<a href="#">Юпитер - это потухшая звезда?</a>
	<p>...</p>
</div>

    
<div class="block1">
	<a href="#">Как выключить компьютер?</a>
	<p>...</p>
</div>
			
    
$('.block1').showHide();

  • Вызов плагина с параметрами.

    В данном примере явно задано имя куки равное cookie-1, внутри блока клик отслеживается на элементе с классом control ; первоначально блоки показаны, мгновенный показ/отображение второго дочернего элемента блока (время анимации 0 мс).

    
<div class="block2">
	<div class="block2_header">
		<h3>Настройки</h3>
		<span class="control"></span>
	</div>
	<p>...</p>
</div>

<div class="block2">
	<div class="block2_header">
		<h3>Настройки 2</h3>
		<span class="control"></span>
	</div>
	<p>...</p>
</div>
			
    
$('.block2').showHide({
	cookieName: 'cookie-1',
	clickElem: '.control',
	visible: true,
	time: 0
});
